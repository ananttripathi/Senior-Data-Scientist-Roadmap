# Beginner Projects

Start here after completing Phase 1-2 basics.

## Suggested Projects
1. **House Price Prediction** — Linear/Polynomial regression
2. **Iris Classification** — Logistic regression, KNN
3. **Titanic Survival** — Classification (Kaggle)
4. **Customer Churn** — Imbalanced data handling
5. **Sentiment Analysis** — Text classification
6. **Customer Segmentation** — K-Means clustering

## For Each Project
Create a folder with:
- `README.md` — Problem, approach, results
- `notebooks/` or `src/` — Your code
- `requirements.txt` — Dependencies

See [Phase 7 Projects](../progress/phase7-projects.md) for full template.
